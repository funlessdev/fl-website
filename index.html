FunLess is a new generation, research-driven serverless platform focused on speed and ease of use.

Get started

Open-source Apache 2.0 License. GitHub v0.7.0

Free and Open Source

FunLess is an open source platform. You can take it and use it as you want, free of charge.

FaaS has never been so easy

Use the CLI to deploy either locally or on a k8s cluster with just one command, and start developing.

Web Assembly Runtimes

Your functions are run as Web Assembly modules. It's lightining fast, with 0 cold start delays.

Elixir & Rust based

We combined the massive concurrency capabilities of Elixir with the speed and safety of Rust, for a new serverless platform.

Scale to Infinity

Build and scale your app with FunLess. Develop functions that will be handled and scaled automatically.
